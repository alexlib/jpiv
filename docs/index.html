JPIV

JPIV is a complete, open source, and platform independent software package for 2D Particle Image Velocimetry (PIV). JPIV is widely used in many fields of research.

Features:

  • multi-pass, multi-grid, FFT based correlation algorithm
  • ensemble correlation / ensemble averaging
  • single pixel ensemble evaluation
  • vector field filtering, outlier detection
  • rectangular interrogation areas possible
  • calculation of vorticity and other derivatives
  • 3d velocity field reconstruction
  • image display (png, tif, pgm, imx, im7)
  • publication ready vector field display
  • vector field graphics export (eps, pdf, emf, svg, png, jpeg, ...)
  • velocity profile extraction, flow estimation
  • vector field statistics
  • region of interest
  • synthetic image generation
  • command line control and scripting via BeanShell
  • batch processing of a list of files as well as directories


Running JPIV

Prerequisite: A Java Runtime Environment (JRE).

  1. Download the zip-archive.
  2. Extract it to your hard disc.
  3. Open a terminal and cd into the new directory.
  4. Execute java -jar JPIV.jar.

If you experience an out of memory error or poor performance, increase the avalable memory: To allow up to four gigabytes, type java -jar -Xmx4G JPIV.jar

JPIV in Research

For me, the most impressive way of showing the capabilities of JPIV, is to check out the creative output of JPIV users on Google Scholar. It is impossible to list all applications here, but to show the universal scope of the software, I've selected a few examples:

  • Maria L. Moriconi estimated the wind speed on Jupiter using JPIV. conference abstract
  • Benjamin Meunier investigated the fanning flow of a fish before spawning with JPIV. https://doi.org/10.1215/21573689-2413017
  • Joseph M. Sherwood investigated red blood cell aggregation and cell-depleted layer characteristics in microchannels with JPIV. https://dx.doi.org/10.1063%2F1.4717755
  • P. Nardone used JPIV to investigate the flow field behind a ship propeller. researchgate.net
  • Mathias Schlenk measured velocity profils in Microjets with JPIV. https://doi.org/10.1021/acs.langmuir.8b00062
  • E. Gustafsson used JPIV to analyse the deformation of a steel sheet during shearing. https://dx.doi.org/10.1186%2Fs40712-016-0067-x
  • Shiqiong Hu tracks the movement of podosomes (structures on animal cell membranes) with JPIV. https://doi.org/10.1091/mbc.e11-01-0086
  • A. L. Bernassau shows with JPIV how mammalian cells move under certain conditions.
  • . researchgate.net
  • J. Vogelsang shows how soil deforms, when a pile is driven into the ground. https://doi.org/10.1007/978-3-319-23159-4_2

Trivia

I developed JPIV in the scope of the work on my PhD thesis about micro-scale blood flow measurements. Since 2004, JPIV is in use for research and education and constantly maintained. In 2020, a collaboration with the OpenPIV project has started. As a first step, a simple GUI has been introduced, with a workflow that resembles JPIV.
