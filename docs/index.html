JPIV

JPIV is a complete, open source, and platform independent software package for 2D Particle Image Velocimetry (PIV).

JPIV is useful for:

  • multi-pass, multi-grid, FFT based correlation algorithm
  • ensemble correlation / ensemble averaging
  • single pixel ensemble evaluation
  • vector field filtering, outlier detection
  • rectangular interrogation areas possible
  • calculation of vorticity and other derivatives
  • 3d velocity field reconstruction
  • image display (png, tif, pgm, imx, im7)
  • publication ready vector field display
  • vector field graphics export (eps, pdf, emf, svg, png, jpeg, ...)
  • velocity profile extraction, flow estimation
  • vector field statistics
  • region of interest
  • synthetic image generation
  • command line control and scripting via BeanShell
  • batch processing of a list of files as well as directories


You have full access to the well documented source code. Contributions are welcome!

You have full access to the core classes and methods of JPIV via simple scripts. Scripting also enables you to integrate your Matlab, Perl, or Python scripts, as well as any other command line tool. Have a look at the scripting intro for more information.

I developed JPIV in the scope of the work on my PhD thesis about micro-scale blood flow measurements. Since 2004, JPIV is in use for research and education and constantly maintained. In 2020, a collaboration with the OpenPIV project has started. As a first step, a simple GUI with a workflow, comparable to that of JPIV has been introduced.
